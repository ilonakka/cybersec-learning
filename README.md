# cybersec-learning

## Root-Me - Challenges réseau
Résolution de challenges d'analyse réseau avec Wireshark.

### Challenges complétés
- FTP - Authentication : analyse de trafic FTP, extraction de credentials
- TELNET - Authentication : analyse de trafic TELNET, TCP Stream
- Twitter authentication : analyse HTTP, décodage Base64

### Outils utilisés
- Wireshark (analyse de captures réseau .pcap)
- Base64 decode
